## **.net8+winform+Antdui制作LOL小助手**

[TOC]



#### 一、事件起因

 本人lol菜鸡一枚，最近天天玩排位模式，有时候自己分路有优势，其他路却突然就炸了，想着看看他们的战绩，一打开就是隐藏的，所以想着有没有工具能够在进游戏的时候就能够查看队友的战绩，去网上一看发现可以调用lol官方的api，那咱就准备捣鼓捣鼓了。

#### 二、筹备工作

  在一番寻找下也发现了有些关于助手的工具，但是大多数都是要收费功能，免费的功能也不是很多，既然如此那咱就自己来做一个助手出来。

  在git上看了挺多相关的项目，没有发现.net做出来的lol助手，作为.net门派的门前扫地小厮，我毅然决然的冲在前面。思来想去，决定使用.net8+winform+Antdui进行开发,Antdui也是能支持winform的一款优秀的ui开源框架
